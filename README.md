# Anime Wind Editor

Anime Wind Editor is an open-source desktop application for creating subtle animations from static illustrations.

Despite its name, the project is not limited to anime-style artwork. It is designed to animate selected parts of any illustration, such as hair, clothing, leaves, shadows, water, or other elements, while keeping the original image intact.

The primary goal is to provide a lightweight and easy-to-use alternative for creating simple animated scenes without the complexity of full 2D animation software.

---

## Project Status

🚧 Early development (MVP)

The project is currently focused on building a solid foundation before adding advanced animation features.

---

## MVP Goals (Version 0.1)

The first public version will focus on a minimal but functional workflow.

Features planned for v0.1:

- Open an image
- Navigate the canvas (zoom and pan)
- Create editable masks
- Save and load projects
- Preview basic mesh deformation
- Export a simple animation (GIF)

The purpose of this release is to validate the core architecture before introducing more advanced tools.

## Project Philosophy

Anime Wind Editor is intentionally developed in small, incremental steps.

Each version is expected to be fully functional before new features are added. The focus is on simplicity, maintainability, and a clean architecture rather than rapid feature growth.
